Sources de données et référentiels

Dans le cadre de mon alternance, Je suis amené à travailler avec des acteurs publiques qui ont pour sources de nombreux référentiels mais aussi qui font appel à leur propre sources de terrain.

Ainsi un catalogue de données permet : - de référencer des données de manière simplifiée ou avancée et de gérer leurs droits d’accès - de rechercher et consulter des jeux de données via des facettes et filtres associés - de faciliter l’accès et la réutilisation des données via une api - d’indexer des catalogues infra-régionaux - de remonter les métadonnées conformes à la Directive Inspire vers le portail national de Géocatalogue - de remonter par organisation les données présentes dans le catalogue qui répondent aux critères « open data » vers le portail national data.gouv
